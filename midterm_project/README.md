# 🌿 植栽日誌與澆水提醒器

## 一、專案簡介
這是一個以植物照護為主題的互動式網頁應用，使用者可以：
- 新增植物名稱、上次澆水日期與照片  
- 系統會自動計算距離上次澆水的天數  
- 若超過七天未澆水，會顯示警示提醒  
- 可一鍵更新「澆水日期」、刪除個別或全部紀錄  
- 支援深色模式與 localStorage 永久保存資料  

此專案結合 **使用者介面設計、JavaScript 動態互動、Bootstrap 響應式布局** 等技術，實現簡潔直覺的植物照護紀錄系統。

---

## 二、使用技術
| 類別 | 技術 |
|------|------|
| 前端結構 | HTML5 |
| 樣式設計 | CSS3、Bootstrap 5 |
| 程式互動 | JavaScript (DOM 操作、事件監聽) |
| 驗證機制 | Constraint Validation API |
| 資料儲存 | localStorage |
| 額外功能 | 深色模式、自動日期更新、即時警示提醒 |

---

## 三、功能特色
1. 🌱 **新增植物紀錄**：輸入名稱、日期、上傳照片並即時顯示。  
2. 💧 **澆水提醒**：自動計算距離上次澆水的天數，超過七天會警示。  
3. 🔄 **更新澆水日期**：點擊「💧澆水」即可更新當前日期。  
4. 🗑️ **刪除與清除功能**：可刪除單筆或一鍵清除全部紀錄。  
5. 🌗 **深色模式切換**：支援主題切換與 localStorage 記錄設定。  
6. 💾 **資料永久保存**：即使重新整理，植物紀錄仍會保留。
