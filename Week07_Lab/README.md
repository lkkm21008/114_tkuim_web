# Week07_Lab - 會員註冊表單

## 功能清單
- 即時驗證（blur 後顯示、input 即時更新）
- 自訂錯誤訊息（中文提示）
- 事件委派：興趣標籤
- submit 攔截 + 第一錯誤聚焦
- 防重送（Loading 狀態）
- 密碼強度條（弱 / 中 / 強 / 非常強）
- localStorage 暫存欄位內容
- 重設按鈕清除所有內容

## 加分項
1.密碼強度條  
2.localStorage 暫存  
3.重設功能  
4.自訂錯誤訊息與視覺強化（紅框樣式）

## 自訂錯誤條件
- Email 格式不符
- 手機非 09 開頭 10 碼
- 密碼不足 8 碼或未含英數
- 確認密碼不一致
- 未選擇興趣
- 未勾選服務條款
